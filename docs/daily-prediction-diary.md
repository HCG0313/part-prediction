## 최근 요약

2026-07-09 장마감 이후 마감 수집과 재학습을 수행했습니다. ECOS 환율, Naver 뉴스/FOMO, Naver 수급, KIS 수급 overlay, pykrx, OpenDART, KIS 마감 스냅샷, 글로벌 보조 데이터가 2026-07-09 기준으로 갱신되었습니다. KIS 마감 스냅샷은 240개 종목 모두 성공했고, KRX 공식 API는 정상 응답이지만 공식 기준일이 2026-07-08로 하루 늦어 보조 데이터로 분리 기록했습니다.

어제 예측과 오늘 실제 결과를 비교하면 예측 Top3는 반도체/전자, 조선/방산, 금융이었고 실제 Top3는 반도체/전자, 2차전지, 통신이었습니다. Top3 겹침은 1/3입니다. 반도체/전자는 후보 보존이 맞았지만, 2차전지와 통신은 행동층에서 낮게 잡혔고 금융은 크게 빗나갔습니다.

오늘 해결한 핵심 문제는 후보 신뢰도와 shadow 모델 검증입니다. 후보 신뢰도 성과 파일을 갱신해 C/D 등급 후보가 실제로 상위권에 들어갔는지 추적했고, 베타 라우터와 딥러닝 MLP shadow 모델을 기존 메인 모델과 비교했습니다. 현재 성능 리더는 기존 메인 모델이고, 성장 가능성 리더는 베타 라우터입니다.

2026-07-10 대상 최신 예측은 반도체/전자, 조선/방산, 2차전지, 통신, 유통/소비 순입니다. 다만 확률 신뢰도는 낮아 반도체/전자는 방어 관찰, 조선/방산과 2차전지는 관망, 통신과 금융은 행동층 관찰 후보로만 기록합니다.
