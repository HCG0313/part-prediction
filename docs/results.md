# 검증 결과

## 해석 기준

이 프로젝트의 검증은 투자 수익률을 확정적으로 보여주는 것이 아니라, 전일 예측이 다음 거래일 실제 섹터 흐름과 얼마나 맞았는지 확인하는 과정입니다.

주요 기준은 다음과 같습니다.

| 기준 | 의미 |
| --- | --- |
| Top N 겹침 | 예측 상위 섹터와 실제 상위 섹터가 얼마나 겹쳤는지 |
| 방향 정확도 | 상승/하락 방향 판단이 맞았는지 |
| 행동 판단 | 핵심 관찰, 보조 관찰, 관망, 회피 우선이 실제 장세와 맞았는지 |
| 모델 학습 포인트 | 틀린 이유를 다음 모델 개선으로 연결했는지 |

## 현재 성능 판단

2026-06-11 장마감 기준 누적 방향 정확도는 48.4%입니다. 아직 예측 모델로 강하다고 말할 수준은 아니지만, 프로젝트의 목적은 단순 적중률 자랑이 아니라 매일 오차를 기록하고 개선하는 것입니다.

현재까지는 다음 경향이 보입니다.

- 급락장에서는 회피 게이트가 어느 정도 유효합니다.
- 뉴스/FOMO 신호만으로는 다음 날 강세를 안정적으로 맞히기 어렵습니다.
- 장중 breadth와 거래대금 가중 수익률을 함께 봐야 주도 섹터 판별이 좋아집니다.
- 급락 후 급반등장에서는 모델이 보수적으로 남아 주도 섹터를 늦게 잡는 문제가 있습니다.

## 개선 목표

| 수집 상태 기반 검증 | API 기준일 지연이 성능 평가를 왜곡하지 않도록 방어 |

## 최신 결론

현재 모델은 "맞히는 모델"이라기보다 "틀린 이유를 누적해 개선하는 모델"에 가깝습니다. 포트폴리오 관점에서는 이 점이 오히려 중요합니다. 단순 결과보다 데이터 수집 문제, 전처리 문제, 모델 판단 오류를 실제 날짜별 사례로 추적하고 있기 때문입니다.
