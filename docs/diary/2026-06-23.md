## 오늘 점검한 문제

오늘은 새 하이브리드 모델을 바로 만들지 않고, 현재 서브 모델들의 생성 시점과 갱신 상태를 먼저 확인했다. 대부분의 서브 모델은 만들어진 지 1~4일 정도밖에 되지 않았고, 장중 반등 분류 모델처럼 오래된 구조도 있었지만 최근 장마감 기준으로 재학습 산출물은 정상 갱신되고 있었다.

따라서 지금 바로 메인 모델을 교체하는 것은 이르다고 판단했다. 대신 현재 V5를 `챔피언 모델`로 유지하고, V6 shadow rank, LGBMRanker, 예상수익률 V2, FOMO blend, panic rebound watch, meta action filter를 `챌린저 모델`로 두어 시간에 따른 성능을 비교하기로 했다.

새 하이브리드 모델은 방향 자체는 유효하지만, 지금 즉시 메인으로 올리면 최근 며칠 성과에 과적합될 수 있다. 그래서 하이브리드 구조는 조금 더 고민하되, 만든다면 처음부터 메인이 아니라 shadow 모델로 운용하는 것이 맞다고 정리했다.
