# 파트 예측

> 뉴스, 검색 관심도, FOMO 기대효과, 환율, 수급, 차트 데이터를 이용해 한국 주식 섹터의 다음 거래일 반응을 예측하는 연구형 프로젝트 기록이다.
> 이 문서는 투자 추천이나 자동매매 지침이 아니라, 모델 개발 과정과 예측 검증을 누적하는 개발 일지다.

## 프로젝트 한 줄 요약

- 한국 주식시장에서 특정 뉴스와 관심도 급증이 FOMO성 매수 압력으로 이어지는지 분석한다.
- 다음 거래일에 강할 가능성이 높은 섹터를 예측한다.
- 현재는 섹터 단위 예측을 중심으로 하고, 각 섹터별 대표 종목을 약 20개까지 확대해 데이터 품질을 높이고 있다.
- 최종 목표는 실제 투자 판단을 대신하는 모델이 아니라, 개인 포트폴리오 의사결정을 보조하는 분석 비서로 발전시키는 것이다.

## 핵심 가설

- 단순 차트만으로는 다음 날 섹터 움직임을 안정적으로 설명하기 어렵다.
- 뉴스 강도, 검색 관심도, 거래대금 증가, 개인·외국인·기관 수급, 환율 충격이 결합될 때 단기 섹터 반응이 더 잘 설명될 수 있다.
- FOMO는 상승 신호이면서 동시에 과열과 반전 위험 신호가 될 수 있다.
- 따라서 모델은 관심도 상승을 무조건 긍정 신호로 보지 않고, 기대효과와 과열 위험을 함께 계산한다.

## 데이터 수집 구조

- KRX Open API
  - 종목 시세, 지수, ETF, 파생상품 일부 데이터를 수집한다.
- pykrx
  - 국내 종목·지수 가격 데이터를 보조 수집한다.
- 한국은행 ECOS
  - 환율 및 거시 변수를 수집한다.
- NAVER 검색 API
  - 뉴스 데이터를 수집한다.
- NAVER DataLab
  - 검색 트렌드 수집을 시도하지만, 현재는 인증 문제로 news proxy 방식을 사용한다.
- OpenDART
  - 공시 데이터를 수집한다.
- 한국투자증권 KIS
  - 실시간·분봉성 차트 수집을 시도한다.
  - 타임아웃 발생 시 Naver Finance realtime fallback을 사용한다.
- Kaggle/Yahoo 기반 글로벌 시장 데이터
  - 해외시장, 원자재, 글로벌 리스크 변수를 보조 변수로 사용한다.
