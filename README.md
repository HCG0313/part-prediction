# 파트 예측

> 뉴스, 검색 관심도, FOMO 기대효과, 환율, 수급, 차트 데이터를 이용해 한국 주식 섹터의 다음 거래일 반응을 예측하는 연구형 프로젝트 기록이다.
> 이 문서는 투자 추천이나 자동매매 지침이 아니라, 모델 개발 과정과 예측 검증을 누적하는 개발 일지다.

> 매일 장마감 후 `데이터 수집 → 예측 결과 검증 → 모델 학습 → 다음 거래일 예측 → 문제점 기록` 순서로 작성한다.  
> 상세 기록은 `docs/diary/` 폴더의 날짜별 파일에 따로 저장한다.
